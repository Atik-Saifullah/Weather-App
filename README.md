# 🌤️ Weather App

A modern, clean, and fully functional Weather Application built to provide real-time weather updates for any location using JavaScript and a live Weather API.

---

## 🚀 Current Status: Completed 🎉
- [x] **HTML5 Structure:** Semantic layout completed.
- [x] **CSS3 Styling:** Responsive design, beautiful layout, and modern UI completed.
- [x] **JavaScript Integration:** Fully connected with Weather API for live, dynamic data fetching.

---

## 📱 Live Features
- ✨ **Real-time Data:** Fetches live weather conditions, temperature, humidity, and wind speed.
- 🔍 **City Search:** Users can search for any city worldwide to get instant weather updates.
- 🎨 **Modern UI/UX:** Clean glassmorphism/minimalist design optimized for Mobile, Tablet, and Desktop screens.

---

## 🛠️ Tech Stack Used

| Technology | Purpose |
| :--- | :--- |
| **HTML5** | Semantic structure of the application |
| **CSS3** | Layout styling and responsiveness |
| **JavaScript (ES6+)** | API Integration, Fetch API, and DOM Manipulation |
