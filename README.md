## Weather App

A modern and clean Weather Application built to provide real-time weather updates for any location. This project is a great showcase of interactive front-end web development.
---

## Current Status: Work In Progress 
- **HTML5 Structure:** Semantic layout completed.
- **CSS3 Styling:** Beautiful layout, and modern UI completed.
- **JavaScript Integration:** API fetching and dynamic DOM updates (Scheduled for tomorrow).

---

##  Features Implemented So Far
-  **Modern UI/UX:** Clean glassmorphism/minimalist design for a premium feel.
-  **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop screens using CSS Flexbox/Grid.
-  **Visual Placeholders:** Beautifully designed cards for temperature, humidity, wind speed, and weather condition icons.
